Mobi → StarDict Converter

Convert Kindle .mobi dictionaries to StarDict format - entirely offline, right in your browser. No installation, no uploads, no DRM removal.

✨ 100 % client-side  ·  No data leaves your device

🔄 Converters

Two converters are available depending on the format of your .mobi file. If you are unsure which one to use, try the KF8 converter first - it works with most modern Kindle dictionaries.

⚡
KF8 · Newer editions

KF8 / Huff Converter

For dictionaries in the modern KF8 format using HUFF/CDIC compression. Works with 2026 and recent editions.

Open Converter →
📜
Mobi 7 · Older editions

PalmDoc Converter

For older Mobi 7 dictionaries that use PalmDoc compression. Works with the 2014 edition and some earlier updates.

Open Converter →

🚀 How to use

1

Pick the right converter

Choose KF8 for newer editions or PalmDoc for older ones. When in doubt, start with KF8.

2

Open the converter page & select your file

Click Open Converter, then pick your purchased .mobi dictionary file from your device.

3

Choose entry style & convert

Select the StarDict entry style that suits your reader and click Convert / Begin.

4

Download & install

Download each generated file, place them together in a new folder inside /dict on your e-reader (e.g. Onyx Boox).

💡 About the project

The main motivation behind this project is to make the excellent Wielki Słownik Angielsko-Polski - a Kindle-only Polish–English dictionary - available on other e-readers like Onyx Boox that support the StarDict format.

The decompression algorithm is inspired by KindleUnpack, but completely rewritten from scratch in JavaScript to run entirely in the browser - no server, no Python, no dependencies.

The repository contains only the conversion tool. The dictionary itself must be purchased separately. This project does not distribute or circumvent any DRM - it only reads the content you already own.
